# Guide utilisateur - Consultation

## Acces
- Compte rattaché au groupe `consultation` (lecture seule).
- Menus : Accueil `/`, Formations `/formations/`, Classes `/formations/classes/`, Reporting `/reporting/`, enquetes (lecture) et Messages (lecture contacts/campagnes si autorisé).

## Taches principales
- Consulter les données (formations, classes, enquetes).
- Explorer les tableaux de bord et exporter les données pour analyse.

## Flows detaillees
1) Parcours lecture  
   - Formations : `/formations/` (statuts).  
   - Classes : `/formations/classes/` puis fiche classe pour voir les apprenants/enquetes liés.  
   - Enquetes : `/presences/`, `/satisfaction-apprenants/`, `/satisfaction-formateurs/`, `/environnement/` (filtrer par classe).
2) Reporting  
   - `/reporting/` : consulter compteurs/top listes, exporter CSV/XLS.
3) Exports  
   - Sur chaque page d’enquete : lien Export CSV.  
   - Messages : `/messages/export/csv/` si autorisé.

## Controles / tests rapides
- Vérifier l’accès en lecture aux pages listées.  
- Télécharger un CSV (presence ou satisfaction) et vérifier son ouverture.  
- Consulter le reporting et exporter CSV/XLS.  
- Confirmer l’absence de boutons d’édition/suppression (lecture seule).
